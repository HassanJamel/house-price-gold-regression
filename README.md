## ğŸ† Kaggle Requirements

### Why is this dataset useful?

It provides 79 explanatory variables describing almost every aspect of residential homes in Ames, Iowa, making it perfect for practicing advanced regression.

### What is the goal?

Predict the final price of each home. For each Id in the test set, you must predict the value of the SalePrice variable.

### Metrics

Submissions are evaluated on **Root-Mean-Squared-Error (RMSE)** between the logarithm of the predicted value and the logarithm of the observed sales price.
